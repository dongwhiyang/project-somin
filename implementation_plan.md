# 프로젝트 소민 V2.2 실무 융합형 원스톱 파이프라인 개편안

사용자 개입을 최소화하고 AI 모델의 강점을 극대화하는 **완전 자동화 '원스톱' 파이프라인**의 최신 버전(V2.2)입니다.

## 📌 주요 업데이트 (V2.2)

1. **단계별 AI 페르소나 및 목표 설정 (System Prompts)**
   - 각 단계마다 구체적인 전문가 페르소나를 부여하여 콘텐츠의 전문성과 일관성을 높입니다.
   - **1단계 (주제 선정)**: 10년 차 파워블로거 및 국토부 전문 자문위원.
   - **2단계 (초안 작성)**: 유명 기술 저널 스타 칼럼니스트.
   - **3단계 (교차 비판)**: 대학교수 및 기술사 (Qwen & Mistral Small 기반).
   - **4단계 (이미지 생성)**: NVIDIA SDXL을 이용한 블로그 삽화 자동 생성.
   - **5단계 (최종 수정)**: 토목공학계 최고 권위자 수석 편집장.

2. **NVIDIA NIM API 기반 고성능 모델 통합**
   - 기존의 일반 모델들을 NVIDIA API를 통한 최신 LLM(Llama 3.1 70B, Mixtral 8x22B 등)으로 교체하여 추론 성능을 극대화합니다.
   - 이미지 생성: `stabilityai/stable-diffusion-xl-base-1.0`을 사용하여 기술적인 삽화를 생성합니다.

3. **원스톱 자동화 및 이미지 자동 삽입**
   - 버튼 클릭 한 번으로 [수집 -> 초안 -> 비판 -> 이미지 생성 -> 최종 수정 -> 튜닝]이 연속 실행됩니다.
   - 생성된 이미지는 본문의 가장 적절한 위치에 마크다운 형태로 자동 삽입됩니다.

## 🛠 워크플로우 상세

### 1단계: 하이브리드 주제 선정 (NVIDIA Llama 3 70B)
- 로컬 기출문제 데이터와 실시간 구글 뉴스 데이터를 조합하여 6개(수험 4, 실무 2)의 주제를 제안합니다.

### 2단계: DeepSeek V4 기술 초안 작성
- 선정된 주제와 수집된 공공 데이터를 바탕으로 1,500~2,000자 내외의 전문적인 초안을 작성합니다.

### 3단계: 듀얼 기술 비판 (Qwen & Mistral Small 대체 모델)
- NVIDIA API를 통해 두 명의 전문가가 기술적 정확성, 가독성, 논리적 허점을 매섭게 비판합니다.
- 이 과정에서 블로그에 필요한 **이미지 생성 프롬프트**를 함께 도출합니다.

### 4단계: 이미지 자동 생성 (NVIDIA SDXL)
- 비판 단계에서 도출된 프롬프트를 바탕으로 최대 3장의 고품질 삽화를 생성하고 로컬 `images/` 폴더에 저장합니다.

### 5단계: DeepSeek 최종 수정 및 이미지 삽입
- 모든 비판 내용과 생성된 이미지 경로를 통합하여 수석 편집장 페르소나가 최종 블로그 글을 완성합니다.

### 6단계: Mistral AI 문체 튜닝
- 최종본을 독자가 읽기 편한 경어체(~해요, ~입니다)로 다듬어 실무 블로그에 최적화합니다.
